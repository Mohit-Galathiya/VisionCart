# VisionCart
E-Commerce Android App using Kotlin &amp; Jetpack Compose
---

## 🚀 Features

### 👤 User App
- 🔐 User Authentication (Firebase)
- 🛍️ Product Catalog (Dynamic Listing)
- ➕ Add to Cart (Quantity Counter, Size Selection)
- ❤️ Wishlist Management
- 🔍 Search Functionality
- 🎯 Banner Display (Offers & Promotions)
- 👤 Edit Profile
- 💳 Razorpay Payment Integration
- ☁️ Firebase Firestore:
  - Product Data Fetching
  - User Data Management
  - Order Management

---

### 🛠️ Admin App (Future Scope)
- 🔐 Admin Authentication
- 📦 Product Management (Add, Update, Delete)
- 🗂️ Category Management
- 🎯 Banner Management
- 📊 Dashboard:
  - Total Users
  - Total Orders
  - Total Products
  - Revenue Overview
- 🌙 Dark Mode Support
- 👤 Admin Profile Management
- 🔔 Push Notifications to Users

- 💉 Dagger Hilt (Dependency Injection)
- 🧭 Navigation Component
- 🖼️ Coil (Image Loading)
- 🔄 Kotlin Serialization
- 📄 Pager (Horizontal Paging UI)
- 🔥 Firebase SDK

---

## 🏗️ Architecture

The project follows **MVVM (Model-View-ViewModel)** architecture:

- **Model** → Data Layer (Firebase / API)
- **ViewModel** → Business Logic
- **View (UI)** → Jetpack Compose UI
